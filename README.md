# 宵鳴き館（縦画面・改善版）

スマホ縦画面向けの短編探索ホラーです。
GitHub Pages にそのまま配置して公開できます。

## この版で改善した点
- 黒い円形の視界制限を削除
- 調べる動作は「調べる」ボタンだけで反応
- 主人公をより人らしい見た目に調整
- ドット絵の見分けやすさを改善
- 次にやることを上部UIと「次の目的」ボタンで常時確認可能
- 子供部屋の謎をかなり分かりやすく調整
- TRUE END に必要な白い花も誘導を追加

## 操作
- 左下スティック: 移動
- 調べる: 正面の対象を調べる
- 歩く / 走る: 移動速度切り替え
- 保存: セーブ
- タイトル: タイトルに戻る

## GitHub Pages 公開方法
1. このZIPを解凍
2. 中身を GitHub リポジトリにアップロード
3. `Settings > Pages` を開く
4. Branch を `main` / folder を `/root` に設定
5. 保存すると公開URLが発行されます
