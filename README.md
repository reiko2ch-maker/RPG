# 宵鳴き館 縦画面版

スマホの縦画面で遊べる探索ホラーの軽量ブラウザゲームです。

## 変更点
- 縦画面前提のUIへ再構成
- 画面を邪魔しない下部操作パネル
- 左下ドラッグスティックで移動
- ゲーム画面タップでも「調べる」可能
- 走るボタンを押しっぱなし式から切り替え式に変更

## 操作
- 左下スティック: 移動
- 調べる: 右の大ボタン or ゲーム画面タップ
- 歩く / 走る: 右下の切り替えボタン
- 保存: 上部の保存ボタン
- 題名: タイトルへ戻る

## GitHub Pages で公開する方法
1. このフォルダ一式をリポジトリに入れる
2. `Settings > Pages` を開く
3. Branch を `main`、フォルダを `/root` か `/docs` に指定
4. 数十秒後に公開URLが発行されます

## ファイル
- `index.html`
- `style.css`
- `main.js`

ローカルでもそのまま開けます。
